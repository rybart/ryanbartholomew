devCodeBar

React | Mocha & Chai | Node.js

devCodeBar was built to solve the hassle of downloading programming assessments from a repo and running the environment yourself. The application allows the test administrator to enter their email, select which tests they want, send a url link to the test taker and receive the results via email once the taker has completed their part. The test taker is able to code and see in real time what they got right and what is still not passing the tests.
